# Privacy Policy (Internal Draft)

## Data Collected
- Account data (email, role, org association)
- Clinical documentation and session events
- Audit metadata (IP, user agent, action records)

## How Data Is Used
- Provide clinical session documentation workflows
- Sync and backup data securely
- Meet legal and compliance obligations

## Data Protection
- Encryption in transit and at rest
- Access controls and role permissions
- Audit logging for PHI access

## Data Sharing
- No sale of PHI
- Sharing only for service operation and legal compliance

## User Rights
- Access and correction requests
- Data export/deletion requests as applicable
